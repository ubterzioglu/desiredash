🔴 Backend Eksikleri (Supabase)

  • Supabase Client Setup
    React projesine Supabase client entegrasyonu
  • Auth Context & Hooks
    Giriş/çıkış, session yönetimi, protected routes
  • RLS Policies (Row Level Security)
    Her tablo için güvenlik kuralları
  • Storage Buckets
    Avatar, logo, etkinlik fotoğrafları için depolama
  • Edge Functions
    Email gönderimi, ödeme webhook, AI entegrasyonu

🟠 Veritabanı Tabloları

  • profiles tablosu
    Kullanıcı profilleri, roller, iletişim bilgileri
  • consultants tablosu
    Danışman detayları, uzmanlık alanları, KYC durumu
  • associations tablosu
    Dernekler, kuruluşlar, topluluklar
  • businesses tablosu
    İşletmeler, claim durumu, sahibi
  • events tablosu
    Etkinlikler, katılımcılar, boost durumu
  • reviews tablosu
    Yorumlar, puanlar, moderasyon durumu
  • favorites tablosu
    Kullanıcı favorileri

🛠️ Mevcut Teknoloji Stack

React 18 TypeScript Vite Tailwind CSS shadcn/ui React Router TanStack Query Zod React Hook Form Vitest

➕ Eklenecek Stack

Supabase PostgreSQL Supabase Auth Supabase Storage Edge Functions Row Level Security

🔮 İleri Seviye (V2+)

React Native Stripe OpenAI API Redis Docker CI/CD Pipeline

⚠️ Güvenlik Kontrol Listesi

  • RLS Policies Aktif
    Tüm tablolar için row level security
  • Input Validasyonu
    Zod ile form ve API validasyonu
  • Rate Limiting
    API istek limitleri, brute force koruması
  • HTTPS/SSL
    Production SSL sertifikası
  • CORS Ayarları
    Domain bazlı CORS kısıtlamaları
